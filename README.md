# discord-reminder-bot

discord-reminder-bot

### Setup

```
// Clone Repository
$ git clone https://github.com/SunstroUS/discord-reminder-bot.git
// Install Dependencies
$ npm install
// Run application
$ npm run test
// Open http://localhost:5000/ in browser
```
